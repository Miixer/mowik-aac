# Komunikator AAC

Prosta strona/aplikacja AAC po polsku: kafelki, budowanie wypowiedzi, synteza mowy, tryb telefonu i synchronizacja własnych kafelków przez kod.

## Publikacja na GitHub Pages

1. Utwórz nowe repozytorium na GitHubie, np. `komunikator-aac`.
2. Wgraj do niego zawartość tego folderu.
3. Wejdź w `Settings` -> `Pages`.
4. W `Build and deployment` wybierz `Deploy from a branch`.
5. Wybierz branch `main` i folder `/root`.
6. Po chwili strona będzie dostępna pod adresem:

```text
https://TWOJ_LOGIN.github.io/komunikator-aac/
```

Jeśli repozytorium nazwiesz dokładnie `TWOJ_LOGIN.github.io`, strona będzie działała pod:

```text
https://TWOJ_LOGIN.github.io/
```
